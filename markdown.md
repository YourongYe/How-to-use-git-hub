# markdown支持HTML语法
上标：<sup>xxx</sup>
下标：<sub>xxx</sub>
函数：&fnof;(x)=x+1
根号：&radic;5
角度符号：30&deg;

更多特殊符号：https://blog.csdn.net/html5_/article/details/21639475
